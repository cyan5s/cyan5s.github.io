Actuator本地docker搭建

Created2025-11-11|Updated2025-11-22|环境搭建
|Post Views:

  • 在开始Actuator未授权漏洞学习之前，为了便于理解actuator，准备在本地使用 Docker Compose 来启动一个简单的 Spring Boot + Actuator 环境

  • 软件安装

    • docker：Docker: Accelerated Container Application Development

      cmd输入docker –version和docker compose version验证

    • Eclipse：Eclipse Downloads | The Eclipse Foundation

      Eclipse IDE for Java Developers

  • 创建 Docker 环境中的 Spring Boot 项目

    • 创建 Spring Boot 项目

    • 创建一个项目目录 spring-boot-docker，在该目录下创建以下两个文件：

      • Dockerfile

        • FROM openjdk:17-jdk-slim
VOLUME /tmp
ARG JAR_FILE=target/*.jar
COPY ${JAR_FILE} app.jar
ENTRYPOINT ["java","-jar","/app.jar"]

  • 构建 Spring Boot 应用

    • 导入项目到 IDE

      • File > Import > Maven > Existing Maven Projects > Browse > 解压后的下载文件 > Fnish

    • 检查

      • 确认 Maven 依赖是否正确加载

        • 在 Package Explorer 中，展开 Maven Dependencies，确保所有需要的依赖都已经加载。如果没有加载，请右键点击项目文件夹，选择 Maven > Update Project，然后点击 OK 来强制更新依赖。

      • 查看 pom.xml 文件，确保有

        • <dependencies>
<dependency>
<groupId>org.springframework.boot</groupId>
<artifactId>spring-boot-starter-web</artifactId>
</dependency>
<dependency>
<groupId>org.springframework.boot</groupId>
<artifactId>spring-boot-starter-actuator</artifactId>
</dependency>
</dependencies>

> @SpringBootApplication
> public class SpringBootDockerApplication {
> public static void main(String[] args) {
> SpringApplication.run(SpringBootDockerApplication.class, args);
> }
> @GetMapping("/")
> public String home() {
> return "Hello, Spring Boot!";
> }
> }
>

      • 确保在 src/main/resources 目录下有一个名为 static 或 public 的文件夹，且有（如 index.html）

        • index.html

          • <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Spring Boot Docker</title>
</head>
<body>
<h1>Welcome to Spring Boot with Docker!</h1>
</body>
</html>
